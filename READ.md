## 项目介绍
这是一个仿《罗小黑战记》官方妖精论坛的同人项目，旨在还原动画中妖精与人类共存的网络社区生态，包含用户系统、帖子发布、权限管理等核心功能。

## 技术栈
- 后端：Python / Flask / Django
- 前端：HTML / CSS / JavaScript
- 数据库：JSON
